Lesson 10 - CSS Border Radius Property

Lesson Objective

The border-radius property is used to create rounded corners on an HTML element.

Syntax

selector{
    border-radius:20px;
}

HTML Code

<div class="box">
Rounded Box
</div>

CSS Code

.box{
    background:#1565C0;
    color:white;
    padding:25px;
    border-radius:20px;
}

JavaScript Code

const box=document.querySelector(".box");

box.onclick=function(){

if(this.style.borderRadius=="60px"){

this.style.borderRadius="20px";

}else{

this.style.borderRadius="60px";

}

}

Live Example

Rounded Box

Click the box to change the border radius.

Explanation

HTML creates the box.


CSS rounds the corners using border-radius:20px;.


JavaScript changes the corners between 20px and 60px when the box is clicked.
