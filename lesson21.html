Lesson 21 - CSS Float Property

Lesson Objective

The float property moves an element to the left or right, allowing surrounding text and other elements to wrap around it.

Syntax

.floatbox{
float:left;
}

HTML Code

<div class="floatbox">CSS</div>

<p>
This text wraps around the floated box.
</p>

CSS Code

.floatbox{
float:left;
margin-right:20px;
}

JavaScript Code

const box=document.querySelector(".floatbox");

box.onclick=function(){

if(this.style.float=="right"){

this.style.float="left";

}else{

this.style.float="right";

}

}

Live Example

CSS

The float property allows text to wrap around an element. This was commonly used for layouts before Flexbox and CSS Grid became popular. Today, it is still useful for wrapping images inside paragraphs and other specific layout situations. Click the blue box to move it from the left side to the right side.

Explanation

HTML creates a box and a paragraph.


CSS floats the box to the left so the paragraph wraps around it.


JavaScript changes the float direction between left and right whenever the box is clicked.
