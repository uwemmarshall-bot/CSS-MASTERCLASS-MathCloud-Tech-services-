Lesson 2 - CSS Background Color Property

Lesson Objective

Learn how to use the background-color property to change the background colour of an HTML element.

Syntax

selector{
    background-color:value;
}

HTML Code

<div class="box">
CSS Background
</div>

CSS Code

.box{
    background-color:orange;
    color:white;
    padding:25px;
}

JavaScript Code

document.querySelector(".box")
.addEventListener("click",function(){

this.style.backgroundColor="green";

});

Live Example

CSS Background

Click the orange box to change its background colour.

Explanation

The HTML creates a box using a div element.


The CSS applies an orange background colour and white text.


The JavaScript changes the background colour from orange to green when the box is clicked.
