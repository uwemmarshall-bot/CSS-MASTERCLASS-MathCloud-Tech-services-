Lesson 23 - CSS Z-Index Property

Lesson Objective

The z-index property controls which positioned element appears in front when two or more elements overlap.

Syntax

.box{
position:absolute;
z-index:2;
}

HTML Code

<div class="box1">Blue Box</div>

<div class="box2">Green Box</div>

CSS Code

.box1{
position:absolute;
z-index:1;
}

.box2{
position:absolute;
z-index:2;
}

JavaScript Code

const box=document.querySelector(".box1");

box.onclick=function(){

if(this.style.zIndex=="3"){

this.style.zIndex="1";

}else{

this.style.zIndex="3";

}

}

Live Example

Blue Box
Green Box

Click the blue box to bring it to the front.

Explanation

HTML creates two overlapping boxes.


CSS uses position:absolute; and different z-index values to control which box appears on top.


JavaScript changes the blue box between z-index:1; and z-index:3; when clicked.
