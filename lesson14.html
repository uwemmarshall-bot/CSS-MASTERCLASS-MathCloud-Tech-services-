Lesson 14 - CSS Position Property

Lesson Objective

The position property controls where an element appears on a webpage. Common values include relative, absolute, fixed, and sticky.

Syntax

.box{
position:relative;
left:50px;
top:20px;
}

HTML Code

<div class="parent">

    <div class="box">
        Position
    </div>

</div>

CSS Code

.parent{
position:relative;
}

.box{
position:relative;
left:0;
top:0;
}

JavaScript Code

const box=document.querySelector(".box");

box.onclick=function(){

if(this.style.left=="120px"){

this.style.left="0px";
this.style.top="0px";

}else{

this.style.left="120px";
this.style.top="50px";

}

}

Live Example

Position

Click the blue box to change its position.

Explanation

HTML creates the container and the box.


CSS uses position:relative so the box can move using left and top.


JavaScript moves the box to a new position whenever it is clicked.

Practice Exercise

  • left:150px;
  • top:30px;

  • position:absolute;

  • position:sticky;
