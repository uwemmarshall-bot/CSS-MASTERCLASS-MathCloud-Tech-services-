Lesson 27 - Advanced Table Design

Lesson Objective

Create a professional result table using table captions, alternating row colors, hover effects, status colors, borders, and responsive design.

HTML Code

<table>
<caption>Student Results</caption>

<tr>
<th>Name</th>
<th>Course</th>
<th>Score</th>
<th>Grade</th>
</tr>

</table>

CSS Code

table{
border-collapse:collapse;
width:100%;
}

tbody tr:hover{
background:#D6ECFF;
}

.pass{
background:#D4EDDA;
}

.fail{
background:#F8D7DA;
}

JavaScript Code

const rows=document.querySelectorAll("tbody tr");

rows.forEach(row=>{

row.onclick=function(){

alert(this.cells[0].innerText);

}

});

Explanation

HTML creates a complete examination result table.


CSS adds alternating row colors, hover effects, borders, captions, and colored status cells.


JavaScript displays the selected student's name when a row is clicked.
