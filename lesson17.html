Lesson 17 - CSS Box Shadow Property

Lesson Objective

The box-shadow property adds a shadow around an HTML element, making it look raised and more attractive.

Syntax

.box{
box-shadow:0 8px 20px rgba(0,0,0,.3);
}

HTML Code

<div class="box">
Box Shadow
</div>

CSS Code

.box{
background:#1565C0;
box-shadow:0 8px 20px rgba(0,0,0,.3);
}

JavaScript Code

const box=document.querySelector(".box");

box.onclick=function(){

if(this.style.boxShadow=="0px 0px 0px transparent"){

this.style.boxShadow="0 8px 20px rgba(0,0,0,.3)";

}else{

this.style.boxShadow="0px 0px 0px transparent";

}

}

Live Example

Box Shadow

Click the box to remove and restore the shadow.

Explanation

HTML creates the box.


CSS adds a shadow using the box-shadow property.


JavaScript removes and restores the shadow whenever the box is clicked.
