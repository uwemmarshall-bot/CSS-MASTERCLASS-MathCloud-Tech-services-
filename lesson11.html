Lesson 11 - CSS Width Property

Lesson Objective

The width property is used to set the horizontal size of an HTML element.

Syntax

selector{
    width:300px;
}

HTML Code

<div class="box">
Width Example
</div>

CSS Code

.box{
    width:300px;
    background:#1565C0;
    color:white;
    padding:20px;
}

JavaScript Code

const box=document.querySelector(".box");

box.onclick=function(){

if(this.style.width=="500px"){

this.style.width="300px";

}else{

this.style.width="500px";

}

}

Live Example

Width Example

Click the blue box to change its width.

Explanation

HTML creates the box.


CSS sets the width of the box to 300px.


JavaScript changes the width between 300px and 500px when the box is clicked.
