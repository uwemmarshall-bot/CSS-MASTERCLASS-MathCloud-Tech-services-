Lesson 20 - CSS Outline Property

Lesson Objective

The outline property adds a line around an element. Unlike a border, an outline does not take up extra space and is often used to highlight selected or focused elements.

Syntax

.box{
outline:4px solid orange;
outline-offset:6px;
}

HTML Code

<div class="box">
Outline Example
</div>

CSS Code

.box{
outline:4px solid orange;
outline-offset:6px;
}

JavaScript Code

const box=document.querySelector(".box");

box.onclick=function(){

if(this.style.outline=="6px dashed red"){

this.style.outline="4px solid orange";

}else{

this.style.outline="6px dashed red";

}

}

Live Example

Outline Example

Click the box to change the outline style.

Explanation

HTML creates the example box.


CSS applies an outline around the box using the outline property.


JavaScript changes the outline between a solid orange outline and a dashed red outline when the box is clicked.
