Lesson 6 - CSS Text Align Property

Lesson Objective

The text-align property is used to align text to the left, center, right, or justify inside an element.

Syntax

selector{
    text-align:center;
}

HTML Code

<div class="box">

    <h2 class="text">
    Welcome Students
    </h2>

</div>

CSS Code

.box{
    border:2px solid #1565C0;
    padding:20px;
}

.text{
    text-align:center;
    color:#1565C0;
    font-size:30px;
}

JavaScript Code

const text=document.querySelector(".text");

text.onclick=function(){

if(this.style.textAlign=="left"){

this.style.textAlign="center";

}else{

this.style.textAlign="left";

}

}

Live Example

Welcome Students

Click the heading to change the text alignment.

Explanation

HTML creates the container and heading.


CSS centers the heading using text-align:center;.


JavaScript changes the alignment between center and left when the heading is clicked.
