Lesson 19 - CSS Cursor Property

Lesson Objective

The cursor property changes the appearance of the mouse pointer when it moves over an HTML element.

Syntax

.box{
cursor:pointer;
}

HTML Code

<div class="box">
Hover Over Me
</div>

CSS Code

.box{
cursor:pointer;
background:#1565C0;
}

JavaScript Code

const box=document.querySelector(".box");

box.onclick=function(){

if(this.style.cursor=="crosshair"){

this.style.cursor="pointer";

}else{

this.style.cursor="crosshair";

}

}

Live Example

Hover Over Me

Click the box to switch between the pointer and crosshair cursor.

Explanation

HTML creates the interactive box.


CSS changes the mouse pointer using the cursor property.


JavaScript changes the cursor style between pointer and crosshair whenever the box is clicked.
