Lesson 26 - CSS Table Styling

Lesson Objective

Create professional HTML tables using CSS properties such as border-collapse, padding, background-color, hover effects, and nth-child().

HTML Code

<table>

<tr>
<th>Name</th>
<th>Course</th>
<th>Score</th>
</tr>

<tr>
<td>John</td>
<td>CSS</td>
<td>95</td>
</tr>

</table>

CSS Code

table{
width:100%;
border-collapse:collapse;
}

th,td{
border:1px solid #ccc;
padding:15px;
}

tr:nth-child(even){
background:#F1F8FF;
}

tr:hover{
background:#E3F2FD;
}

JavaScript Code

const rows=document.querySelectorAll("tr");

rows.forEach(row=>{

row.onclick=function(){

alert("You clicked a table row.");

}

});

Explanation

HTML creates the table structure.


CSS styles the table, adds borders, spacing, alternating row colors, and hover effects.


JavaScript detects when a table row is clicked.
