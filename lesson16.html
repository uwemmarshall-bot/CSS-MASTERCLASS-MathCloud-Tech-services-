Lesson 16 - CSS Opacity Property

Lesson Objective

The opacity property controls the transparency of an HTML element. The value ranges from 0 (completely transparent) to 1 (fully visible).

Syntax

.box{
opacity:0.5;
}

HTML Code

<div class="box">
Opacity
</div>

CSS Code

.box{
opacity:1;
background:#1565C0;
}

JavaScript Code

const box=document.querySelector(".box");

box.onclick=function(){

if(this.style.opacity=="0.3"){

this.style.opacity="1";

}else{

this.style.opacity="0.3";

}

}

Live Example

Opacity

Click the blue box to change its transparency.

Explanation

HTML creates the box.


CSS sets the opacity to 1, making the box fully visible.


JavaScript changes the opacity between 1 and 0.3 when the box is clicked.
