Lesson 4 - CSS Font Family Property

Lesson Objective

The font-family property is used to change the style or typeface of text on a webpage.

Syntax

selector{
    font-family: Arial;
}

HTML Code

<h1 class="text">
Learning CSS
</h1>

CSS Code

.text{
    font-family:"Courier New", monospace;
    font-size:35px;
    color:#1565C0;
}

JavaScript Code

document.querySelector(".text")
.addEventListener("click",function(){

this.style.fontFamily="Georgia";

});

Live Example

Learning CSS

Click the heading to change the font family.

Explanation

The HTML creates the heading.


The CSS changes the font family to Courier New.


The JavaScript changes the font family to Georgia when the heading is clicked.

Practice Exercise
