Lesson 5 - CSS Font Weight Property

Lesson Objective

The font-weight property is used to make text lighter or bolder.

Syntax

selector{
    font-weight:bold;
}

HTML Code

<h1 class="text">
Welcome to CSS
</h1>

CSS Code

.text{
    font-size:36px;
    font-weight:bold;
    color:#1565C0;
}

JavaScript Code

document.querySelector(".text").onclick=function(){

this.style.fontWeight="300";

}

Live Example

Welcome to CSS

Click the heading to change the font weight.

Explanation

HTML creates the heading.


CSS makes the heading bold using font-weight:bold;.


JavaScript changes the font weight to a lighter value when the heading is clicked.

Practice Exercise

  • 300
