Lesson 3 - CSS Font Size Property

Lesson Objective

Learn how to use the font-size property to increase or decrease the size of text.

Syntax

selector{
    font-size:value;
}

HTML Code

<h1 class="text">
Welcome Students
</h1>

CSS Code

.text{
    font-size:40px;
    color:#1565C0;
}

JavaScript Code

document.querySelector(".text")
.addEventListener("click",function(){

this.style.fontSize="60px";

});

Live Example

Welcome Students

Click the heading to increase the font size.

Explanation

The HTML creates a heading using the h1 element.


The CSS sets the text size to 40px.


The JavaScript changes the font size to 60px when the heading is clicked.
