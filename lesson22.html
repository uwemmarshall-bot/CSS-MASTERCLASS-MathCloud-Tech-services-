Lesson 22 - CSS Clear Property

Lesson Objective

The clear property specifies whether an element should move below floated elements. It is commonly used with the float property.

Syntax

.clearbox{
clear:left;
}

HTML Code

<div class="floatbox">Float</div>

<div class="clearbox">
Clear Element
</div>

CSS Code

.floatbox{
float:left;
}

.clearbox{
clear:left;
}

JavaScript Code

const clearBox=document.querySelector(".clearbox");

clearBox.onclick=function(){

if(this.style.clear=="none"){

this.style.clear="left";

}else{

this.style.clear="none";

}

}

Live Example

Float
Click Me To Toggle Clear

Explanation

HTML creates a floated box and another box below it.


CSS uses clear:left; so the green box starts below the floated element.


JavaScript switches between clear:left; and clear:none; when the green box is clicked.

Practice Exercise
