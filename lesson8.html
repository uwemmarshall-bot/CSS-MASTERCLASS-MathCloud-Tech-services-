Lesson 8 - CSS Margin Property

Lesson Objective

The margin property creates space outside an element. It separates one element from another.

Syntax

selector{
    margin:20px;
}

HTML Code

<div class="box">
Margin Example
</div>

CSS Code

.box{
    background:#1565C0;
    color:white;
    padding:20px;
    margin:20px;
    border-radius:8px;
}

JavaScript Code

const box=document.querySelector(".box");

box.onclick=function(){

if(this.style.margin=="60px"){

this.style.margin="20px";

}else{

this.style.margin="60px";

}

}

Live Example

Margin Example

Click the blue box to increase or decrease the margin.

Explanation

HTML creates the box.


CSS adds 20px of space outside the box using the margin property.


JavaScript changes the margin between 20px and 60px when the box is clicked.
