Lesson 7 - CSS Padding Property

Lesson Objective

The padding property creates space between the content of an element and its border.

Syntax

selector{
    padding:20px;
}

HTML Code

<div class="box">
Welcome Students
</div>

CSS Code

.box{
    background:#1565C0;
    color:white;
    padding:20px;
    text-align:center;
}

JavaScript Code

const box=document.querySelector(".box");

box.onclick=function(){

if(this.style.padding=="50px"){

this.style.padding="20px";

}else{

this.style.padding="50px";

}

}

Live Example

Welcome Students

Click the blue box to increase or decrease the padding.

Explanation

HTML creates the box.


CSS adds 20px of space inside the box using the padding property.


JavaScript changes the padding between 20px and 50px whenever the box is clicked.
