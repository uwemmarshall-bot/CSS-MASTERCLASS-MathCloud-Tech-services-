Lesson 24 - CSS Visibility Property

Lesson Objective

The visibility property controls whether an element is visible or hidden while still keeping its original space on the webpage.

Syntax

.box{
visibility:hidden;
}

HTML Code

<div class="box">
Visibility Example
</div>

CSS Code

.box{
visibility:visible;
}

JavaScript Code

const box=document.querySelector(".box");

box.onclick=function(){

if(this.style.visibility=="hidden"){

this.style.visibility="visible";

}else{

this.style.visibility="hidden";

}

}

Live Example

Visibility Example

Click the blue box to hide and show it.

Explanation

HTML creates the example box.


CSS displays the box using visibility:visible;.


JavaScript switches between visibility:visible; and visibility:hidden; whenever the box is clicked.

Practice Exercise

Try these values:

  • visibility:visible;
  • visibility:hidden;
  • Compare visibility:hidden; with display:none;.
  • Create two boxes and hide one using the visibility property.
  • Observe that the hidden element still occupies space.
