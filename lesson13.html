Lesson 13 - CSS Background Color Property

Lesson Objective

The background-color property is used to change the background colour of an HTML element.

Syntax

selector{
background-color:blue;
}

HTML Code

<div class="box">
Background Colour Example
</div>

CSS Code

.box{
background:#1565C0;
color:white;
padding:30px;
text-align:center;
}

JavaScript Code

const box=document.querySelector(".box");

box.onclick=function(){

if(this.style.background=="tomato"){

this.style.background="#1565C0";

}else{

this.style.background="tomato";

}

}

Live Example

Background Colour Example

Click the box to change its background colour.

Explanation

HTML creates the box.


CSS gives the box a blue background using the background-color property.


JavaScript changes the background colour between blue and tomato whenever the box is clicked.
