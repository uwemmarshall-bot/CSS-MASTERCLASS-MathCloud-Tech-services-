Lesson 15 - CSS Overflow Property

Lesson Objective

The overflow property controls what happens when content is too large to fit inside an element.

Syntax

.box{
overflow:hidden;
}

HTML Code

<div class="box">

This is a long paragraph used to demonstrate how the CSS overflow property works when there is too much content inside a fixed height container.

</div>

CSS Code

.box{
height:120px;
overflow:hidden;
}

JavaScript Code

const box=document.querySelector(".box");

box.onclick=function(){

if(this.style.overflow=="scroll"){

this.style.overflow="hidden";

}else{

this.style.overflow="scroll";

}

}

Live Example

CSS Overflow is one of the most useful properties in web design. It controls whether hidden content should be clipped, shown, scrolled or displayed automatically. Click this box to switch between hidden and scroll so you can see the difference.

Click inside the box to change the overflow property.

Explanation

HTML creates the content box.


CSS uses overflow:hidden; to hide extra content.


JavaScript changes the overflow between hidden and scroll whenever the box is clicked.
