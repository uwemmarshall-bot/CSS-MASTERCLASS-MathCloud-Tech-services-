Lesson 9 - CSS Border Property

Lesson Objective

The border property adds a border around an HTML element. You can change its width, style and colour.

Syntax

selector{
    border:4px solid blue;
}

HTML Code

<div class="box">
Border Example
</div>

CSS Code

.box{
    border:4px solid #1565C0;
    padding:20px;
    text-align:center;
}

JavaScript Code

const box=document.querySelector(".box");

box.onclick=function(){

if(this.style.border=="6px dashed red"){

this.style.border="4px solid #1565C0";

}else{

this.style.border="6px dashed red";

}

}

Live Example

Border Example

Click the box to change the border style.

Explanation

HTML creates the box.


CSS gives the box a blue solid border.


JavaScript changes the border between a blue solid border and a red dashed border when the box is clicked.
