Lesson 1 - CSS Color Property

Lesson Objective

Learn how to use the color property to change the colour of text on a webpage.

Syntax

selector{
    color:value;
}

HTML Code

<h1 class="title">
Welcome to CSS
</h1>

CSS Code

.title{
    color:blue;
    font-size:35px;
}

JavaScript Code

document.querySelector(".title")
.addEventListener("click",function(){

this.style.color="red";

});

Live Example

Welcome to CSS

Click the heading below to change its colour.

Explanation

The HTML creates the heading.


The CSS changes the text colour from the default black to blue.


The JavaScript waits until the heading is clicked, then changes the colour from blue to red.
