Lesson 25 - CSS Overflow-X and Overflow-Y Properties

Lesson Objective

The overflow-x property controls horizontal scrolling, while the overflow-y property controls vertical scrolling.

Syntax

.box{
overflow-x:auto;
overflow-y:hidden;
}

HTML Code

<div class="box">

This is a very long line of text that continues across the screen so that you can test the horizontal scrollbar created by the overflow-x property.

</div>

CSS Code

.box{
overflow-x:auto;
overflow-y:hidden;
white-space:nowrap;
}

JavaScript Code

const box=document.querySelector(".box");

box.onclick=function(){

if(this.style.overflowY=="scroll"){

this.style.overflowY="hidden";

}else{

this.style.overflowY="scroll";

}

}

Live Example

This is a very long line of text that demonstrates the CSS overflow-x property. Continue scrolling horizontally to read the remaining content and click this box to enable or disable the vertical scrollbar using the overflow-y property.

Scroll horizontally and click the box to change the vertical overflow.

Explanation

HTML creates the content container.


CSS enables horizontal scrolling using overflow-x:auto; and hides vertical scrolling using overflow-y:hidden;.


JavaScript switches the vertical overflow between hidden and scroll when the box is clicked.

Practice Exercise

Try these values:

  • overflow-x:hidden;
  • overflow-x:scroll;
  • overflow-y:auto;
  • overflow-y:visible;
  • Increase the height and width of the box.
