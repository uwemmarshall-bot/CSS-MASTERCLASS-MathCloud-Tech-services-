Lesson 18 - CSS Text Shadow Property

Lesson Objective

The text-shadow property adds a shadow effect to text, making headings and titles more attractive.

Syntax

.text{
text-shadow:3px 3px 6px gray;
}

HTML Code

<h1 class="text">
CSS Text Shadow
</h1>

CSS Code

.text{
color:#1565C0;
text-shadow:3px 3px 6px rgba(0,0,0,.4);
}

JavaScript Code

const text=document.querySelector(".text");

text.onclick=function(){

if(this.style.textShadow=="none"){

this.style.textShadow="3px 3px 6px rgba(0,0,0,.4)";

}else{

this.style.textShadow="none";

}

}

Live Example

CSS Text Shadow

Click the heading to remove and restore the text shadow.

Explanation

HTML creates the heading.


CSS applies a shadow to the text using the text-shadow property.


JavaScript removes and restores the text shadow whenever the heading is clicked.
